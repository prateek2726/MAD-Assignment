# Mobile Application Development Assignment

This repository contains four Android applications developed as part of the Mobile Application Development (MAD) course assignment. Each project is contained in its own directory.

---

## 📁 Project Directory

| Folder Name | Question | Description |
| :--- | :--- | :--- |
| **MADAssignmentCurrencyConverter** | Q1 | Currency converter (INR, USD, JPY, EUR) with Light/Dark theme. |
| **MADAssignmentMediaPlayer** | Q2 | Audio player and URL video streamer with full controls. |
| **MADAssignmentSensorApp** | Q3 | Real-time data for Accelerometer, Light, and Proximity. |
| **MADAssignmentPhotoGallery** | Q4 | Gallery app with Camera integration and Image management. |

---

## 🚀 Application Details

### Q1: Currency Converter & Theme Switcher
* **Functionality:** Converts between INR, USD, JPY, and EUR.
* **Key Features:** Includes a Settings page to toggle between Light and Dark Mode.

### Q2: Media Player (Audio & Video)
* **Capabilities:** Plays local audio and streams video content via URL.
* **Controls:** Open File, Open URL, Play, Pause, Stop, and Restart.

### Q3: Sensor Data Reader
* **Sensors Monitored:** Accelerometer, Light Sensor, and Proximity Sensor.

### Q4: Camera & Photo Gallery Pro
* **Camera:** Take photos and save them to a chosen folder.
* **Gallery:** View images in a GridView, view metadata, and delete with confirmation.
